Colors and Fonts

Fonts and colors,With CSS, colors can be specified in different ways: Color Names, HTML5 Color Names, Hexadecimal and RGB.

The following fonts are the best web safe fonts for HTML and CSS. Most web browsers will have these fonts by default: Arial (sans-serif) Helvetica (sans-serif) Georgia (serif)

Sometimes you only want to style specific elements of a type instead of all elements of one type. Up until this point, the only styling we have been able to do is styling by element, which means that every element of a particular kind has the same style.. Classes allow for a group of elements to have their own unique style. They allow web developers to treat groups of elements they want styled differently than other elements of the same type
